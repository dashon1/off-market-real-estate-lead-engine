TEMPLATE 1: Python Scraper - Tax Deed Extraction

Use this script to extract tax deed listings from county portals. Customize the CSS selectors based on your county's page structure.

""" Tax Deed Scraper Template Customize selectors for your specific county portal """ import requests from bs4 import BeautifulSoup import pandas as pd from datetime import datetime import time import random class TaxDeedScraper: def __init__(self, county_name, base_url): self.county_name = county_name self.base_url = base_url self.leads = [] def fetch_page(self, url, retries=3): """Fetch page with error handling""" headers = { 'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36' } for attempt in range(retries): try: response = requests.get(url, headers=headers, timeout=30) if response.status_code == 200: return response.content else: print(f"Status code {response.status_code}, retrying...") except Exception as e: print(f"Error: {e}, retrying...") time.sleep(random.uniform(2, 5)) # Random delay return None def parse_tax_deeds(self, html): """Parse tax deed listings - customize selectors""" soup = BeautifulSoup(html, 'html.parser') # CUSTOMIZE THESE SELECTORS for your county # Common patterns - adjust based on actual page structure property_rows = soup.select('table.listings-table tbody tr') # OR use: soup.select('.property-item') # OR use: soup.select('div[data-type="deed"]') for row in property_rows: try: lead = { 'source': 'tax_deed', 'scrape_date': datetime.now().strftime('%Y-%m-%d'), 'county': self.county_name, 'owner_name': self.clean_text(row.select_one('.owner-name, .grantee')), 'property_address': self.clean_text(row.select_one('.property-address, .address')), 'tax_id': self.clean_text(row.select_one('.tax-id, .parcel')), 'amount_due': self.clean_text(row.select_one('.amount-due, .tax-amount')), 'auction_date': self.clean_text(row.select_one('.auction-date, .sale-date')), } if lead['property_address']: self.leads.append(lead) except Exception as e: print(f"Error parsing row: {e}") continue return self.leads def clean_text(self, element): """Clean extracted text""" if element: return element.get_text(strip=True) return '' def save_results(self, filename='tax_deeds.csv'): """Save collected leads to CSV""" if self.leads: df = pd.DataFrame(self.leads) df.to_csv(filename, index=False) print(f"Saved {len(self.leads)} records to {filename}") return filename return None # USAGE EXAMPLE # scraper = TaxDeedScraper("Miami-Dade", "https://taxcollector.county.gov") # html = scraper.fetch_page("https://taxcollector.county.gov/tax-deeds") # if html: # scraper.parse_tax_deeds(html) # scraper.save_results("daily_tax_deeds.csv")

  • Test selectors one at a time to ensure they capture the right data
  • Save the script and run it daily using Task Scheduler (Windows) or Cron (Mac/Linux)
TEMPLATE 2: Lead Scoring Script

Apply this algorithm to automatically score and prioritize your leads based on motivation indicators.

""" Lead Scoring Algorithm Scores leads 1-25 based on motivation factors """ import pandas as pd def score_leads(df): """ Apply scoring algorithm to lead dataframe Required columns: source, owner_state, amount_due, value_gap_pct, is_out_of_state, date_found, absentee_owner """ scores = [] for idx, row in df.iterrows(): score = 0 # 1. SOURCE TYPE SCORING (max 5 points) source_scores = { 'tax_deed': 5, 'pre-foreclosure': 4, 'probate': 3, 'inherited': 3, 'auction': 2, 'divorce': 2 } source = str(row.get('source', '')).lower().strip() score += source_scores.get(source, 1) # 2. GEOGRAPHIC LOCATION (max 5 points) if row.get('is_out_of_state', False): score += 5 elif row.get('owner_state', 'UNKNOWN') != 'UNKNOWN': # In-state but could be remote score += 2 # 3. FINANCIAL PRESSURE (max 5 points) try: amount = float(str(row.get('amount_due', '0')).replace('$', '').replace(',', '')) if amount > 20000: score += 5 elif amount > 10000: score += 4 elif amount > 5000: score += 3 elif amount > 0: score += 1 except: pass # 4. EQUITY OPPORTUNITY (max 5 points) try: gap = float(row.get('value_gap_pct', 0)) if gap > 50: score += 5 elif gap > 30: score += 4 elif gap > 15: score += 3 elif gap > 0: score += 1 except: pass # 5. ABSENTEE/OUT-OF-STATE OWNER (max 3 points) if row.get('absentee_owner', False): score += 3 # 6. TIME FACTOR (max 2 points) try: days = (pd.to_datetime('today') - pd.to_datetime(row.get('date_found', datetime.now()))).days if days > 90: score += 2 elif days > 30: score += 1 except: pass scores.append(score) df['motivation_score'] = scores # Categorize into priority tiers df['priority'] = pd.cut( df['motivation_score'], bins=[0, 8, 14, 20, 30], labels=['Low', 'Medium', 'High', 'Hot'] ) # Sort by score descending df = df.sort_values('motivation_score', ascending=False) return df # USAGE # df = pd.read_csv('your_leads.csv') # scored_df = score_leads(df) # scored_df.to_csv('scored_leads.csv', index=False) # print(f"Hot leads: {len(scored_df[scored_df['priority'] == 'Hot'])}") # print(f"High leads: {len(scored_df[scored_df['priority'] == 'High'])}")

TEMPLATE 5: Lead Qualification Script

Use this script when calling responding leads to qualify them properly.

Opening (1 minute)

"Hi [Name], this is [Your Name] calling. I'm reaching out because we help homeowners who want to sell their property quickly and easily. I saw that [property address] came up in my research and wanted to check if you're still considering a sale. Is now a good time to chat for just a few minutes?"

Situation Assessment (3 minutes)

"Can you tell me a little bit about your situation? What's making you want to sell [property address]?" - Listen for: urgency, emotional triggers, timeline pressure "If you don't mind me asking, how long have you owned the property?" - Listen for: how long they've dealt with any issues "Have you had any major repairs or updates recently?" - Listen for: what they know about condition

Property Details (2 minutes)

"Can you describe the current condition of the property? Are there any major systems that need attention - roof, HVAC, plumbing?" "Have you had any offers from other investors or agents?" - If yes: "What were those offers? What made you not accept?" - This helps understand their expectations

Timeline & Terms (2 minutes)

"What's your timeline looking like? When do you need to close by?" "Are there any loans or liens on the property that need to be paid off?" "What do you need to walk away with from the sale to make this work for you?" - Write down their minimum number

Closer (1 minute)

"I appreciate you sharing all of this with me. Based on what you've told me, I'd like to look at the property and put together a framework for you. Would it be okay if I scheduled a quick walkthrough this week?" "If the numbers make sense for both of us, I can close quickly - typically 2-3 weeks. Does that timeline work for you?" "Let me get my calendar set up and I'll text you to confirm a time."

Red Flags to Watch For

  • Unrealistic price expectations with no flexibility
  • Vague timeline (no urgency mentioned)
  • Still hasn't decided if they want to sell
  • More interested in your credentials than the property
  • Not willing to share basic property details
Note: If a lead fails to provide basic information (address, condition, timeline), they're likely not motivated enough to transact. Focus your time on leads who engage actively.
