1 Identifying Your County Data Portal

The foundation of your off-market lead engine is accessing public data that most investors ignore. Government portals publish tax deeds, probate filings, and pre-foreclosure notices daily—but most investors never check them because they require manual effort. Your job is to automate that effort.

Step 1.1: Locate Your County's Public Data Sources

Not all counties publish data the same way. Start by identifying which of these three sources your county uses:

  1. Clerk of Court — Handles court records including probate filings, lis pendens, and foreclosure proceedings
  2. Comptroller or Tax Collector — Maintains the List of Lands Available (tax deeds) and back tax records
  3. County Recorder — Keeps property transfer records, deed filings, and ownership history

Step 1.2: Find Your County's Specific Portal

Search for "[Your County] clerk of court public records" or "[Your County] tax collector list of lands available." Most counties have one of these formats:

  • Dedicated public records search portal (often via "Xsoft" or "ONbase" software)
  • FTP or downloadable CSV/Excel files of tax deed listings
  • PDF publication updated weekly or monthly
  • Real-time search interface for court filings
Pro Tip: If your county has a "List of Lands Available" page, bookmark it. This is typically updated weekly and contains properties going to tax sale. Check it every Monday morning for new additions.

2 Setting Up Your Scraping Infrastructure

Once you know where your data lives, you need to set up automated collection. This section walks you through two approaches: Python (for technical users) and n8n (for no-code users).

Approach A: Python Scraper Setup (Technical)

If you're comfortable with code, use Python with BeautifulSoup for static pages or Playwright for dynamic/JavaScript-loaded pages.

Prerequisites:

  • Python 3.8+ installed
  • pip package manager
  • Basic understanding of HTML structure

Installation:

# Install required packages pip install requests beautifulsoup4 pandas selenium webdriver-manager # For dynamic pages (JavaScript rendered) pip install playwright playwright install chromium

Basic Scraper Template:

import requests from bs4 import BeautifulSoup import pandas as pd from datetime import datetime import time class PublicDataScraper: def __init__(self, county_name): self.county_name = county_name self.data = [] def fetch_tax_deeds(self, url): """Fetch tax deed listings from county portal""" headers = { 'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36' } response = requests.get(url, headers=headers) soup = BeautifulSoup(response.content, 'html.parser') # Customize selectors based on your county's page structure rows = soup.select('table.properties-list tr') for row in rows: property_data = { 'source': 'tax_deed', 'scrape_date': datetime.now().strftime('%Y-%m-%d'), 'county': self.county_name, 'owner_name': row.select_one('.owner-name').text.strip(), 'property_address': row.select_one('.address').text.strip(), 'tax_id': row.select_one('.tax-id').text.strip(), 'amount_due': row.select_one('.amount-due').text.strip(), 'auction_date': row.select_one('.auction-date').text.strip() } self.data.append(property_data) return self.data def fetch_probate_filings(self, url): """Fetch probate case listings""" headers = { 'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36' } response = requests.get(url, headers=headers) soup = BeautifulSoup(response.content, 'html.parser') # Customize selectors for probate docket cases = soup.select('.case-row') for case in cases: case_data = { 'source': 'probate', 'scrape_date': datetime.now().strftime('%Y-%m-%d'), 'county': self.county_name, 'case_number': case.select_one('.case-number').text.strip(), 'decendent_name': case.select_one('.decedent').text.strip(), 'filing_date': case.select_one('.filing-date').text.strip(), 'attorney': case.select_one('.attorney').text.strip() if case.select_one('.attorney') else 'N/A' } self.data.append(case_data) return self.data def save_to_csv(self, filename): """Export collected data to CSV""" df = pd.DataFrame(self.data) df.to_csv(filename, index=False) print(f"Saved {len(self.data)} records to {filename}") # Usage Example scraper = PublicDataScraper("Miami-Dade") scraper.fetch_tax_deeds("https://taxcollector.county.gov/tax-deeds") scraper.fetch_probate_filings("https://clerk.county.gov/probate-dockets") scraper.save_to_csv("daily_leads.csv")

3 Data Cleaning Protocols

Raw scraped data is messy. Before you can score and contact leads, you need to filter out noise and focus on genuinely motivated sellers. This week transforms your raw list into an actionable pipeline.

Step 3.1: Filter Out Institutional Buyers

The first cleaning pass removes corporate entities that buy at auction but don't sell to individual investors:

Exclusion Patterns:

  • Bank names (Chase, Wells Fargo, Bank of America, etc.)
  • Trust companies (XYZ Trust, [Name] Family Trust)
  • REIT identifiers (REIT, Real Estate Investment Trust)
  • LLC patterns (* LLC, * Limited Liability Company)
  • Government entities (County, State, HUD, VA)
  • National lenders (Freedom Mortgage, Quicken Loans, etc.)

Python Filter Script:

import pandas as pd def filter_institutional(df): """Remove institutional buyers from lead list""" # Patterns to exclude exclude_patterns = [ 'BANK', 'TRUST', 'LLC', 'LTD', 'INC', 'CORP', 'MORTGAGE', 'HUD', 'VA', 'FANNIE', 'FREDDIE', 'NATIONSTAR', 'WELLS', 'CHASE', 'BANK OF AMERICA', 'FREEDOM MORTGAGE', 'OCWEN', 'BAYVIEW', 'SELENE', 'U S BANK', 'US BANK', 'HSBC', 'CITIBANK' ] # Create exclusion pattern pattern = '|'.join(exclude_patterns) # Filter out matches (case-insensitive) mask = ~df['owner_name'].str.upper().str.contains(pattern, na=False) return df[mask].copy() # Usage clean_leads = filter_institutional(raw_leads) print(f"Removed {len(raw_leads) - len(clean_leads)} institutional buyers") print(f"Remaining: {len(clean_leads)} potential seller leads")

Step 3.2: Identify Out-of-State Owners

Out-of-state owners are prime targets because they:

  • Can't easily manage distant properties
  • Often have inherited properties they want to liquidate
  • Face higher transaction costs when managing remotely
  • Are more motivated to sell quickly

State Detection Method:

import re def extract_state_from_address(address): """Extract state abbreviation from mailing address""" # Common patterns for state at end of address patterns = [ r',\s*([A-Z]{2})\s+\d{5}', # ", CA 90210" r',\s*([A-Z]{2})\s+\d{5}-\d{4}', # ", NY 10001-1234" r'\b([A-Z]{2})\b$', # State at end "Los Angeles CA" ] for pattern in patterns: match = re.search(pattern, str(address)) if match: return match.group(1) return 'UNKNOWN' def flag_out_of_state(df, target_state='FL'): """Flag owners with mailing addresses outside target market""" df['owner_state'] = df['mailing_address'].apply(extract_state_from_address) df['is_out_of_state'] = df['owner_state'] != target_state return df # Usage leads_with_state = flag_out_of_state(clean_leads, target_state='FL') out_of_state = leads_with_state[leads_with_state['is_out_of_state'] == True] print(f"Found {len(out_of_state)} out-of-state owners in {target_state} market")

Step 3.3: Cross-Reference with GIS Data

Gravity (GIS) mapping databases contain property characteristics that help identify high-potential deals. Many counties publish assessor data with:

  • Assessed value vs. market value gaps
  • Building square footage and lot size
  • Year built and building condition codes
  • Owner-occupancy status
  • Sales history and price trends

Finding Free GIS Data:

  1. Search "[Your County] GIS portal" or "[Your County] property appraiser data"
  2. Look for "Downloads" or "Public Data" sections
  3. Many counties offer CSV or Excel exports of property characteristics
  4. Some integrate directly with Google Sheets via add-ons

GIS Enrichment Script:

import pandas as pd def enrich_with_gis(leads_df, gis_data_path): """Merge scraped leads with GIS property data""" # Load GIS data gis_df = pd.read_csv(gis_data_path) # Assuming both have a common identifier (parcel number or address) enriched = leads_df.merge( gis_df[['parcel_id', 'assessed_value', 'market_value', 'sqft', 'year_built', 'last_sale_price', 'last_sale_date']], left_on='tax_id', right_on='parcel_id', how='left' ) return enriched def calculate_deal_potential(enriched_df): """Score properties based on GIS data""" # Value gap score (higher gap = more potential) enriched_df['value_gap'] = ( enriched_df['market_value'] - enriched_df['assessed_value'] ) enriched_df['value_gap_pct'] = ( enriched_df['value_gap'] / enriched_df['assessed_value'] * 100 ) # Price per square foot opportunity enriched_df['price_per_sqft'] = ( enriched_df['last_sale_price'] / enriched_df['sqft'] ) # Time since last sale (older = potential motivated seller) enriched_df['years_since_sale'] = ( pd.to_datetime('today') - pd.to_datetime(enriched_df['last_sale_date']) ).dt.days / 365 return enriched_df # Usage enriched_leads = enrich_with_gis(clean_leads, 'county_gis_data.csv') scored_leads = calculate_deal_potential(enriched_leads)
Pro Tip: Properties with high value gaps (market value 50%+ above assessed) often indicate tax protests, inheritance situations, or properties that haven't been updated in county records—good buying opportunities.

import pandas as pd def score_leads(df): """Apply scoring algorithm to lead list""" scores = [] for idx, row in df.iterrows(): score = 0 # Time on market (from public records) # Using days since scraped as proxy days_found = (pd.to_datetime('today') - pd.to_datetime(row['date_found'])).days if days_found > 365: score += 5 elif days_found > 180: score += 3 elif days_found > 90: score += 1 # Out-of-state owner if row.get('is_out_of_state', False): score += 5 elif row.get('owner_state', 'UNKNOWN') != 'UNKNOWN': score += 1 # Source type scoring source_scores = { 'tax_deed': 5, # Highest motivation 'pre-foreclosure': 4, 'probate': 3, 'auction': 2 } score += source_scores.get(row.get('source', '').lower(), 1) # Amount owed (higher debt = higher motivation) try: amount_owed = float(str(row.get('amount_due', '0')).replace('$', '').replace(',', '')) if amount_owed > 10000: score += 5 elif amount_owed > 5000: score += 3 elif amount_owed > 0: score += 1 except: pass # Value gap (if available) try: gap_pct = float(row.get('value_gap_pct', 0)) if gap_pct > 50: score += 5 elif gap_pct > 25: score += 3 elif gap_pct > 0: score += 1 except: pass # Out-of-state absentee owner flag if row.get('absentee_owner', False): score += 3 scores.append(score) df['motivation_score'] = scores # Categorize df['priority'] = pd.cut( df['motivation_score'], bins=[0, 10, 18, 25, 100], labels=['Low', 'Medium', 'High', 'Hot'] ) return df # Usage scored_leads = score_leads(enriched_leads) print(scored_leads['priority'].value_counts())

5 Messaging Frameworks

The difference between getting ignored and getting a response often comes down to how you frame your message. Motivated sellers don't want to be sold—they want to solve a problem.

Step 5.1: Understanding Seller Pain Points

Before writing any message, internalize these common motivations:

  • Probate sellers — Overwhelmed by legal process, want to close quickly, don't want to manage property
  • Tax-delinquent owners — Facing pressure from county, need to clear back taxes, embarrassed about situation
  • Absentee owners — Can't manage distant property, costly to maintain, tired of dealing with tenants or empty homes
  • Inherited property owners — Don't want the property, live far away, unclear what to do
  • Divorce or life-change situations — Need fast sale to move on, emotionally drained

Step 5.2: Core Message Principles

  1. Lead with empathy, not pitch — Acknowledge their situation before offering help
  2. Focus on them, not you — "I help people who..." not "I buy houses"
  3. Reduce friction, not create it — Make responding easy and low-commitment
  4. Be specific, not generic — Reference their actual situation when possible
  5. Create value before asking — Offer useful information, not just a sales pitch

Step 5.3: The Five Outreach Channels

Channel 1: Direct Mail (Letters)

Highest response rates for motivated sellers. Physical mail stands out in a digital world.

Key elements:

  • Handwritten envelope (or handwritten-style printing)
  • Personal letter (one page, 250-300 words)
  • Self-mailer brochure option for follow-up
  • Return envelope included

Timing: Send first letter Day 1, follow-up Day 7, final letter Day 14

Channel 2: Cold Text/SMS

High open rates (98% for texts) but requires compliance. Best for immediate response situations.

Key elements:

  • Personalized first line (their name or property address)
  • Short message (under 160 characters for single SMS)
  • Clear CTA (reply Y, call, or visit link)
  • Compliance disclaimer when required

Timing: Send between 10 AM - 6 PM local time, avoid Mondays and weekends

Channel 3: Cold Calling

Highest conversion when done right, but most avoided. Focus on building rapport, not selling.

Key elements:

  • Research notes visible on screen before call
  • Natural conversation opener (not a script)
  • Focus on listening, not talking
  • Set clear next step before hanging up

Timing: Best days: Tuesday-Thursday, times: 9-11 AM, 2-4 PM

Channel 4: Email

Lowest cost, but also lowest response rates. Use for nurture sequences, not cold outreach.

Key elements:

  • Catchy subject line (their address or situation as hook)
  • Short paragraphs (3-4 sentences max)
  • Single clear CTA
  • Mobile-optimized

Timing: Tuesday-Thursday mornings, test different send times

Channel 5: RVM (Ringless Voicemail)

Mass outreach that feels personal. Drop voicemail without ringing, low cost per contact.

Key elements:

  • 30-second max script
  • First name mention
  • Property address reference
  • Callback number

Timing: Send after mail piece, before calling attempt

Compliance Reminder: Always verify TCPA, FTC, and state regulations before any text, call, or email campaign. Opt-out requests must be honored immediately. Consider using compliance-focused platforms like PhoneWagon, CallRail, or similar for SMS and calling.

8 Deal Analysis & Offer Structuring

Every deal opportunity requires structured analysis before making an offer. This prevents emotional decisions and ensures profitability.

Step 8.1: The MAO Calculator (Maximum Allowable Offer)

Your MAO is the highest price you can pay while still making a profit after all costs.

def calculate_mao(): """ Maximum Allowable Offer Calculator Fill in your numbers below """ # INPUTS after_repair_value = 200000 # ARV - What property will be worth after repairs repair_costs = 25000 # Estimated repairs holding_costs = 3000 # Expected holding costs (taxes, insurance, utilities) selling_costs = 12000 # Closing costs + agent fees (typically 8-10% of sale) desired_profit = 20000 # Your target profit # CALCULATIONS total_costs = repair_costs + holding_costs + selling_costs + desired_profit max_offer = after_repair_value - total_costs # OUTPUT print(f"After Repair Value (ARV): ${after_repair_value:,}") print(f"Total Costs: ${total_costs:,}") print(f" - Repairs: ${repair_costs:,}") print(f" - Holding: ${holding_costs:,}") print(f" - Selling: ${selling_costs:,}") print(f" - Profit: ${desired_profit:,}") print(f"") print(f"MAXIMUM ALLOWABLE OFFER: ${max_offer:,}") # Save this as your starting point, not your final offer # Offer 10-15% below MAO for negotiation room opening_offer = max_offer * 0.85 print(f"Suggested Opening Offer (15% below MAO): ${opening_offer:,}") calculate_mao()

Step 8.2: The Six-Point Deal Analysis

Before making any offer, evaluate these six factors:

  1. Location Quality — Neighborhood, schools, amenities, job market
  2. Property Condition — Major systems (roof, HVAC, foundation), cosmetic needs
  3. Owner Motivation — Urgency level, flexibility on price and terms
  4. Market Conditions — Days on market, comparable sales, inventory levels
  5. Exit Strategy Clarity — Wholesale, retail, rental, flip? What's the plan?
  6. Risk Factors — Title issues, code violations, environmental concerns

9 Your Ongoing System

This 30-day sprint establishes your system. To maintain a steady flow of off-market deals, continue these habits:

Daily (5-10 minutes)

  • Check new leads from automated scraper
  • Respond to any incoming inquiries immediately
  • Log all outreach activity

Weekly (1-2 hours)

  • Review and score new leads
  • Send follow-up sequences to active leads
  • Update pipeline status for all active leads
  • Analyze what's working (track metrics)

Monthly (half-day)

  • Deep review of closed deals (what worked, what didn't)
  • Adjust scoring weights based on conversion data
  • Optimize outreach messaging based on response rates
  • Explore new data sources to expand lead pool
Long-Term Success: Off-market investing is a volume game. The more leads you process, the more deals you find. Build systems that scale, and focus on relationships. Most of your best deals will come from sellers who referred you to their friends and family.
