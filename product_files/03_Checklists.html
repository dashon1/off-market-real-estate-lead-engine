OFF-MARKET LEAD ENGINE
Complete Verification Checklists

Use these checklists to ensure every step in your off-market system is complete before moving to the next phase.

WEEK 1 CHECKLIST

Data Infrastructure & Scraper Setup

Step 1: County Data Portal Identification

Identified Clerk of Court records portal — URL: ____________________
Identified Tax Collector or Comptroller data source — URL: ____________________
Identified County Recorder or GIS portal — URL: ____________________
Documented all data sources in spreadsheet — Source count: _______
Confirmed update frequency for each source (daily/weekly/monthly)

Step 2: Scraping Infrastructure Setup

Installed Python 3.8+ OR created n8n account
Installed required packages (requests, beautifulsoup4, pandas, selenium/playwright)
Tested connection to first data source — Success: [ ] Yes [ ] No
Successfully extracted sample data — Records collected: _______
Set up automated daily scraping schedule — Time: _______ AM/PM
Configured error notifications (email alert if scraper fails)

Step 3: Database Configuration

Created lead database (Google Sheets/Airtable/SQLite)
Added all required fields — Lead ID, Source, Owner Name, Address, Date Found, Score, Status
Set up data validation rules (prevent duplicate entries, required fields)
Connected scraper output to database — Automation working: [ ] Yes [ ] No
Created backup system (daily export or version history)
Week 1 Sign-Off: Before proceeding to Week 2, confirm that your scraper is running daily without supervision, and your database is receiving new leads automatically.

WEEK 2 CHECKLIST

Data Cleaning & Lead Scoring

Step 4: Institutional Buyer Filter

Created list of institutional buyer patterns (banks, trusts, LLCs, government entities)
Ran filter against all raw leads — Removed: _______ records
Manually reviewed false positives (names that match but aren't institutions)
Verified clean list maintains quality data

Step 5: Geographic Targeting

Extracted state/location from mailing addresses
Identified out-of-state owners — Count: _______
Flagged absentee owners — Count: _______
Verified address accuracy (no obviously invalid addresses)

Step 6: GIS Data Enrichment

Located county GIS or property appraiser data download
Downloaded or connected to GIS dataset
Merged GIS data with lead list — Match rate: _______%
Calculated value gap (ARV vs assessed)
Added price-per-square-foot calculations
Flagged properties with high value gap potential — Count: _______

Step 7: Scoring System Implementation

Defined scoring weights for each factor
Built automated scoring script/formula
Applied scores to all leads — Range: _______ to _______
Categorized leads into priority tiers (Hot/High/Medium/Low)
Reviewed top-scored leads for accuracy
Documented scoring methodology for future calibration
Week 2 Sign-Off: Before proceeding to Week 3, you should have a cleaned, scored list with Hot and High priority leads ready for outreach.

WEEK 3 CHECKLIST

Omnichannel Outreach Activation

Step 8: Outreach Channel Setup

Direct Mail: Created letter templates for each lead type
Direct Mail: Set up mail service account (SendPro, Lob, etc.)
Direct Mail: Verified mailing addresses are deliverable
SMS/Text: Signed up for TCPA-compliant SMS platform
SMS/Text: Created text message templates
Calling: Set up business phone number (local or toll-free)
Calling: Prepared call script and qualification questions
RVM: Recorded 30-second voicemail drop

Step 9: Compliance Verification

Reviewed TCPA requirements for your state
Verified DNC (Do Not Call) list scrubbing process
Created opt-out handling procedure
Set up internal do-not-call list in CRM

Step 10: Outreach Sequence Launch

Prioritized top 50-100 leads for first outreach batch
Personalized message content (reference property address, owner name)
Scheduled Day 1 mail drop
Loaded SMS list for Day 7 text
Scheduled follow-up sequence (Day 3 RVM, Day 5 call, Day 10 mail, etc.)

Step 11: Response Tracking Setup

Created response tracking fields (response date, channel, quality)
Set up notification system for incoming responses
Established daily response review routine
Week 3 Sign-Off: Before proceeding to Week 4, you should have sent initial outreach to top priority leads and be actively tracking responses.

WEEK 4 CHECKLIST

Pipeline Management & Offer Structuring

Step 12: Pipeline Review

Reviewed all incoming responses — Response count: _______
Moved responded leads to appropriate pipeline stage
Scheduled follow-up calls with warm leads
Documented conversation notes in CRM

Step 13: Lead Qualification

Completed qualification call for each responding lead
Verified property details (address, condition, ownership)
Confirmed motivation level and timeline
Discussed pricing expectations
Qualified leads moved to "Under Review" stage — Count: _______

Step 14: Deal Analysis

For each qualified lead, gathered:
ARV estimate (comparable sales or county data)
Repair estimate (walk-through or photo review)
Holding cost estimate
Calculated Maximum Allowable Offer (MAO)
Determined opening offer price (MAO minus negotiation room)

Step 15: Offer Submission

Prepared written offer document (using template from Templates file)
Including proof of funds or pre-approval letter
Submitted Offer #1 — Property: ____________________ | Price: $_______
Submitted Offer #2 — Property: ____________________ | Price: $_______
Submitted Offer #3 — Property: ____________________ | Price: $_______
Tracking offer responses (accepted/declined/pending)
30-Day Goal Check: You should have submitted at least 3 offers by Day 30. If you have fewer, focus on moving more leads through the pipeline quickly.

ONGOING SYSTEM CHECKLIST

Weekly & Monthly Maintenance

Weekly Tasks (1-2 hours)

Review new leads from scraper — New leads this week: _______
Score and categorize new leads
Follow up with non-responsive leads (continue sequence)
Update pipeline status for all active leads
Analyze response rates by channel
Document what's working and what isn't

Monthly Tasks (half-day)

Review closed deals (wins and losses)
Adjust scoring weights based on conversion data
Optimize outreach messaging
Explore new data sources (expand lead pool)
Review and optimize budget allocation
Set goals for next month
Success Habits: Consistent execution beats occasional intensity. Spend 15 minutes daily on lead management rather than 2 hours once a week. Your pipeline stays warm, and you catch opportunities faster.

Use these checklists weekly to maintain system integrity. Update and customize as you learn what works in your specific market.
